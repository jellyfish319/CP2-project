기본적인 프로그래밍 개발 방향을 다음과 같이 잡았습니다.

1. 각 클래스 별 수행할 메서드 모두 구현
2. 1.에서 구현한 메서드를 활용하여 클래스 별로 사용할 GUI의 패널 구현 및 이벤트 처리
3. 패널간 관계 정리
4. 메인 클래스에서 최종 프로그램 동작 로직 구현

이러한 메커니즘으로 Database와 DatabaseGUI, Main 파일에 로그인 화면을 띄우는 패널을 최종적으로 띄우는 것을 시험삼아 구현하였습니다.
참고해주시고 궁금한 점 있으시면 물어봐주시고 아이디어 있으시면 부담갖지 마시고 내주시면 감사하겠습니다.

java파일들을 보시면 아시겠지만 Database 객체에 객체 정보가 업데이트되어야 하기 때문에 Database 구현 부분이 오래 걸릴 것이라 생각합니다.
그래서 일단 Database 구현 1명, Show 구현 1명, Customer 구현 1명, Payment, Seat 구현 1명으로 일을 분배하되 먼저 구현을 완료하신 분 께서는 Database 구현을 도와주시는게 어떨까 하는 생각을 하고 있습니다.

파일도 다 확인하셨으면 카톡으로 다 확인했다고 연락주시고 어떤 파트를 맡아서 구현하고 싶은지 보내주시면 되겠습니다.
